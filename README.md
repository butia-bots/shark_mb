# SHARK Mobile Base

SHARK is a open-source and cost-effective mobile base solution for service robotics, developed for [BUTI√ÅBots](https://fbot.vercel.app/) domestic robot (DoRIS) in [Robocup@Home](https://athome.robocup.org/) league.

Its development is also part of the Brazilian and Uruguayan collaborative research project, funded by the RoboCup Federation with the aim to enable and stimulate new researchers in this field.
